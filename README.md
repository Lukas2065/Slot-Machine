# Slot-Machine
This project aims to create a desktop toy slot machine (No actual gambling involved!!!).

## v1.0 Specifics
The project is currently breadboard based. Using the Arduino Nano, SSD1315 0.96 OLED screen and a potentiometer. When the user twists the potentiometer (future lever) and then returns it to its resting position, the screen plays a slot game.

A PCB has been designed and sent off to be made.

## Future Aims
Make the slot animations cleaner. 3D print a shell to hold the Arduino and pcb and a lever.
